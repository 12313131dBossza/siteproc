Follow these steps to capture professional screenshots from your Vercel deployment and add them to your WordPress landing page.

1 Open Your Vercel Site

Navigate to your deployed SiteProc application on Vercel.

https://your-siteproc.vercel.app

Make sure you're logged in to see the full dashboard and features.

2 Prepare Your Browser

  • Use Google Chrome or Firefox for best results
  • Press F12 to open Developer Tools
  • Click the device toggle icon (or press Ctrl+Shift+M)
  • Set resolution to 1920 x 1080 for desktop screenshots
  • Set resolution to 375 x 667 for mobile screenshots

3 Capture Screenshots

Method 1: Chrome DevTools (Recommended)

  1. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
  2. Type "screenshot"
  3. Select "Capture full size screenshot"
  4. Image downloads automatically

Method 2: Snipping Tool (Windows)

  1. Press Windows + Shift + S
  2. Select area to capture
  3. Click to capture
  4. Save as PNG
üí° Pro Tip: Take screenshots at different states (empty, with data, loading, etc.) to show various use cases.

5 Optimize Your Screenshots

Recommended Sizes:

  • Dashboard/Hero: 1200 x 800px
  • Feature Screenshots: 800 x 600px
  • Mobile Screenshots: 375 x 667px

Optimization Steps:

  1. Crop images to desired size using photo editor
  2. Visit tinypng.com or squoosh.app
  3. Upload and compress images (reduce to < 150KB)
  4. Download compressed images
  5. Rename descriptively: dashboard-overview.png

6 Upload to WordPress

Option A: WordPress Media Library

  1. Go to WordPress Admin ‚Üí Media ‚Üí Add New
  2. Upload all screenshots
  3. Note the image IDs or URLs

Option B: Theme Directory (Recommended)

  1. Create folder: wp-content/themes/siteproc-landing/images/
  2. Upload screenshots via FTP or file manager
  3. Images accessible at: get_template_directory_uri() . '/images/filename.png'
